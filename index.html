Juniverse

Juniverse is a game that seeks to investigate how cubical game worlds can promote curiosity and exploratory behaviour. Research suggests that hidden information causes uncertainty which eventually leads to the player behave in a more exploratory manner. We challenged the natural shape of the cube consisting of only 6 sides by extending this fact. We have suggested a way to implement an infinite amount of sides on the cube making it comparable to more traditional flat terrain based game worlds. It was found that the cube with 16 sides made participants spend more time exploring each side compared to a more regular game world.

  • Unity
  • C#

Video game exploring darkness

This game production was aimed at exploring how a completely dark environment can affect players frustration level. Based on the frustration level from each player between the levels, the amount of light was dynamically adjusted with an implemented algorithm. The simple game mechanics and features of the game made it possible to focus on the dark environment and its affect on frustration.

  • Unity
  • C Sharp

Transforming drawings into 3D game maps

The Map generator is based on image processing techniques in order to translate colors and shapes into playable 3D game worlds. A web cam was used in a start screen to capture drawings from users and generate the map they were about to play themselves. Using traditional image processing techniques, the colors were coded into terrain data and shapes placed on the map were translated into objectives. In this prototype users were able to draw forests, mountains and rivers, as well as start position, end goal, and objects on the map.

  • C Sharp
  • Unity
  • Image Processing

Gesture Controlled Audio Effects

This project came to life from watching a live session video of Nick Cave and the bad seeds. It was noticed that songwriters like Nick Cave often use hand gestures and motions to express alterings in musical performance of a band. This thought ignited the idea of capturing such gestures and actually transforming these into certain audio effects in order to explore it as a tool for band leaders and composers. The gestures were captured using the microsoft kinect 360 and the digital signal processing tool Pure Data was used to analyse and translate the data into meaningful signals for a delay, a reverb and volume.

  • Pure Data
  • Microsoft Kinect
  • Audio Processing
  • DSP

Music application for deaf people

Can deaf people make music?. It turns out that deaf people indeed enjoy music and can interact with it in their own way. So we came up with an application that could facilitate the collaboration between deaf and hearing musicians in making music together. The solution is rather simple, however the apporach is novel. The project took ground in the fact that deaf people often feel music and we suggested an additional visual appearance. We created a feeling of the music with colors, and created visual feedback with objects animating to their corresponding sound.
The projected was peer reviewed and accepted as an official paper on the international conference NIME (New interfaces for musical expressions)

  • Java
  • Processing

Custom built midi controller

The midi controller was built for a fellow musician and friend, in order to be integrated in his live setup. The artist wanted to be able to interact more physically with his music for a more interesting and immersive visual appearance. The midi controller uses a infrared sensor to detect distance of objects above. The distance is translated into meaningful midi signals, which can be mapped to whatever the artist wants it to.

  • Arduino
  • Midi
  • Physical Interface

The Last Leper

The Last Leper is a game made focusing on novel concepts and premise. The game challenges the regular genre of puzzle and exploration games by introducing a new way of challenging the player by requiring the player to manage the health on limbs in order to prevent them to fall off. The game was designed as a proof of concept and was therefore not designed as a complete game with immersive aesthetics as such.

  • Unity
  • C#
